XL Script Converter

 

Instalasi

- Install Python terutama Python 3.10.6
- Unzip xl-convert.zip
- Masuk ke directory xl-convert
- Run source xl-convert/bin/activate untuk mengaktifkan virtual environment
 

Converting Script to BQ:

Cara penggunaan:
python xl-convert.py -q <nama-file-sql> -r regex-definition.rgx -o <nama-file-output>

- nama-file-sql, file yang berisi kode SQL script yang akan diconvert
- nama-file-output, file yang berisi hasil konversi

Contoh:

python xl-convert.py -q script.sql -r regex-definition.rgx -o output.sql

Replace Nama Tabel

Program ini untuk menambahkan nama project di tabel-tabel yang ada di script dengan mencarinya di daftar nama-nama tabel yang ada di BigQuery XL yang bisa diakses oleh user devo.adhis. 

Cara penggunaan:
python replace-query-tables.py -q <nama-sql-file> -s xl-alltables.txt -o output.sql
- nama-file-sql, file yang berisi kode SQL script yang akan diconvert
- nama-file-output, file yang berisi hasil konversi

Tambahan:
xl-alltables.txt berisi list nama-nama tabel yang ada di BigQuery XL yang bisa diakses oleh user devo.adhis.
Untuk mengupdate datanya:
- Login sebagai user devo.adhis dengan menjalankan gcloud auth application-default login
- Jalankan gcp-get-projects-and-tables.py > xl-alltables.txt
